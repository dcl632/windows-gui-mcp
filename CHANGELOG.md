# Changelog

All notable changes to this project will be documented here.

## Unreleased

- Added PyPI Trusted Publishing workflow and release documentation.
- Added contributor, security, pull request, and issue-reporting guidance.

## 0.1.0 - 2026-06-03

- Initial public release of `windows-gui-mcp`.
- Added MCP stdio server entrypoint with `--help` and `--version`.
- Added UI Automation-first tools for window discovery, focusing, tree dumps,
  element lookup, action verification, screenshots, waits, OCR/image fallback,
  and trace-to-script generation.
- Added Notepad and Calculator examples.
- Added MIT license, package metadata, GitHub Actions CI, and release artifacts.
